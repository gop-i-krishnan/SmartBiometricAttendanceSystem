PRO TIPS FOR BEGINNERS:
1. Always disconnect power when wiring
2. Double-check colors match the diagram
3. LEDs only work one way (long leg = positive)
4. Button can go either way (we're using pull-up)
5. Use breadboard for easy prototyping
6. Test each component one at a time

TROUBLESHOOTING HELP:
If something doesn't work:
1. Check all GND connections
2. Verify 3.3V power at components
3. Ensure no wires are loose
4. Confirm correct pin numbers in code
